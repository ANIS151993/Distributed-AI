# IEEE Paper Package

This folder contains the complete IEEE-style manuscript source and generated supporting materials.

## Files

- `IEEE_Distributed_AI_Ensemble.tex`: Main paper source (IEEE conference format)
- `references.bib`: Bibliography database
- `tables/`: Auto-generated LaTeX result tables from benchmark outputs
- `figures/`: Generated charts and benchmark figures used in the manuscript

## Compile

If your environment has TeX Live with IEEEtran:

```bash
cd paper
pdflatex IEEE_Distributed_AI_Ensemble.tex
bibtex IEEE_Distributed_AI_Ensemble
pdflatex IEEE_Distributed_AI_Ensemble.tex
pdflatex IEEE_Distributed_AI_Ensemble.tex
```

Output:

- `IEEE_Distributed_AI_Ensemble.pdf`
